Organ Donation Management System is a desktop application developed in Python that streamlines the process of managing organ donors and recipients. The system utilizes Object-Oriented Programming principles, SQLite database management, and a user-friendly Tkinter GUI to provide an efficient platform for healthcare organizations.

Key Features:
Secure Login Authentication
Donor Registration and Management
Recipient Registration and Management
Automated Organ Matching Algorithm
Blood Group Compatibility Checking
Urgency-Based Priority Queue Processing
Match History Tracking
Dashboard Analytics and Statistics
Blood Group Distribution Pie Chart
PDF Report Generation
SQLite Database Integration

Technologies Used:
Python
Tkinter (GUI)
SQLite Database
Object-Oriented Programming (OOP)
Matplotlib
ReportLab
Heap Queue (Priority Queue)

Project Objective:
The primary goal of this project is to automate donor-recipient management, improve matching efficiency, reduce manual errors, and provide a structured platform for organ donation and transplantation record management.
